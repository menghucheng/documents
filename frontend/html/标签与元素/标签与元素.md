# HTML 标签与元素

## 1. 标签与元素的关系

HTML 标签（Tag）是用于标记内容的代码，通常成对出现：
- **开始标签**：如 `<p>`
- **结束标签**：如 `</p>`
- **内容**：位于开始标签和结束标签之间
- **元素**：开始标签 + 内容 + 结束标签，如 `<p>这是一个段落</p>`

有些标签是自闭合的（单标签），不需要结束标签，如 `<img>`, `<br>`, `<input>` 等。

## 2. 常用 HTML 标签

### 2.1 文档结构标签

| 标签 | 描述 | 示例 |
|------|------|------|
| `<html>` | HTML 文档的根元素 | `<html lang="zh-CN">...</html>` |
| `<head>` | 文档头部，包含元数据 | `<head>...</head>` |
| `<title>` | 文档标题 | `<title>页面标题</title>` |
| `<body>` | 文档主体，包含可见内容 | `<body>...</body>` |
| `<meta>` | 元数据，如字符集、视口等 | `<meta charset="UTF-8">` |
| `<link>` | 引用外部资源 | `<link rel="stylesheet" href="style.css">` |
| `<script>` | 引用或定义脚本 | `<script src="script.js"></script>` |
| `<style>` | 定义内部样式 | `<style>...</style>` |
| `<base>` | 页面所有链接的基准 URL | `<base href="https://example.com">` |

### 2.2 文本内容标签

#### 2.2.1 标题标签

HTML 提供了 6 级标题，从 `<h1>` 到 `<h6>`，`<h1>` 是最高级别的标题，`<h6>` 是最低级别的标题。

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
```

### 2.3 链接标签

#### 2.3.1 基本链接

```html
<a href="https://www.example.com">链接文本</a>
```

#### 2.3.2 链接属性

| 属性 | 描述 | 示例 |
|------|------|------|
| `href` | 链接目标 URL | `href="https://www.example.com"` |
| `target` | 链接打开方式 | `target="_blank"` |
| `title` | 鼠标悬停提示 | `title="访问示例网站"` |
| `rel` | 链接与当前文档的关系 | `rel="noopener noreferrer"` |
| `download` | 下载链接 | `download="filename"` |

#### 2.3.3 链接类型

```html
<!-- 在新标签页打开链接 -->
<a href="https://www.example.com" target="_blank" rel="noopener noreferrer">新标签页打开</a>

<!-- 下载链接 -->
<a href="file.pdf" download="document.pdf">下载文件</a>

<!-- 锚点链接 -->
<a href="#section1">跳转到 section1</a>
...
<h2 id="section1">Section 1</h2>

<!-- 邮箱链接 -->
<a href="mailto:example@example.com">发送邮件</a>

<!-- 电话链接 -->
<a href="tel:+123456789">拨打电话</a>

<!-- SMS 链接 -->
<a href="sms:+123456789?body=Hello">发送短信</a>
```


| 属性 | 描述 | 示例 |
|------|------|------|

#### 2.4.3 响应式图像

```html
<!-- 使用 srcset 和 sizes -->
<img 
    src="image-800w.jpg" 
    srcset="image-400w.jpg 400w, image-800w.jpg 800w, image-1200w.jpg 1200w" 
    sizes="(max-width: 600px) 400px, (max-width: 900px) 800px, 1200px" 
    alt="响应式图片" 
>

<!-- 使用 picture 元素 -->
<picture>
    <source media="(max-width: 768px)" srcset="small.jpg">
    <source media="(max-width: 1200px)" srcset="medium.jpg">
    <img src="large.jpg" alt="响应式图片">
</picture>
```

## 3. 标签分类

### 3.1 按显示类型分类

#### 3.1.1 块级元素（Block-level Elements）

块级元素的特点：
- 独占一行空间
- 宽度默认是父容器的 100%
- 可以设置宽度、高度、内外边距
- 可以包含行内元素和其他块级元素

常见块级元素：
- `<div>`, `<p>`, `<h1>`-`<h6>`
- `<ul>`, `<ol>`, `<li>`, `<dl>`, `<dt>`, `<dd>`
- `<table>`, `<form>`, `<header>`, `<footer>`
- `<section>`, `<article>`, `<nav>`, `<aside>`
- `<address>`, `<blockquote>`, `<pre>`

#### 3.1.2 行内元素（Inline Elements）

行内元素的特点：
- 不独占一行，和其他行内元素在同一行显示
- 宽度由内容决定
- 不能设置宽度、高度（可以设置内边距和边框，但不会影响布局）
- 只能包含文本和其他行内元素

常见行内元素：
- `<span>`, `<a>`, `<strong>`, `<em>`
- `<i>`, `<b>`, `<u>`, `<s>`
- `<code>`, `<kbd>`, `<samp>`, `<var>`
- `<sup>`, `<sub>`, `<br>`, `<hr>`
- `<img>`, `<input>`, `<select>`, `<textarea>`（这些是行内块元素）

#### 3.1.3 行内块元素（Inline-block Elements）

行内块元素的特点：
- 不独占一行，和其他行内元素在同一行显示
- 可以设置宽度、高度、内外边距
- 可以包含文本和其他行内元素

常见行内块元素：
- `<img>`, `<input>`, `<select>`, `<textarea>`
- `<button>`, `<label>`, `<fieldset>`
- `<iframe>`, `<embed>`, `<object>`

### 3.2 按语义分类

#### 3.2.1 语义化标签

语义化标签有明确的含义，能够清晰地描述其包含内容的类型：
- `<header>`, `<nav>`, `<main>`, `<footer>`
- `<article>`, `<section>`, `<aside>`
- `<figure>`, `<figcaption>`, `<time>`
- `<h1>`-`<h6>`, `<p>`, `<blockquote>`
- `<form>`, `<input>`, `<label>`

#### 3.2.2 非语义化标签

非语义化标签没有明确的含义，主要用于布局：
- `<div>`：通用块级容器
- `<span>`：通用行内容器

## 4. 标签属性

### 4.1 全局属性

全局属性可以应用于所有 HTML 元素：

| 属性 | 描述 | 示例 |
|------|------|------|
| `id` | 元素唯一标识符 | `id="header"` |
| `class` | 元素类名 | `class="container"` |
| `style` | 内联样式 | `style="color: red;"` |
| `title` | 元素描述信息 | `title="提示文本"` |
| `lang` | 元素内容语言 | `lang="zh-CN"` |
| `dir` | 文本方向 | `dir="rtl"` |
| `accesskey` | 元素快捷键 | `accesskey="k"` |
| `tabindex` | 元素在 tab 键顺序中的位置 | `tabindex="1"` |
| `contenteditable` | 元素内容是否可编辑 | `contenteditable="true"` |
| `spellcheck` | 元素内容是否拼写检查 | `spellcheck="true"` |
| `hidden` | 元素是否隐藏 | `hidden` |
| `draggable` | 元素是否可拖动 | `draggable="true"` |
| `dropzone` | 元素作为拖放目标时的行为 | `dropzone="copy"` |
| `data-*` | 自定义数据属性 | `data-user-id="123"` |
| `role` | ARIA 角色 | `role="button"` |
| `aria-*` | ARIA 属性 | `aria-label="关闭"` |

### 4.2 自定义数据属性

使用 `data-*` 属性可以为元素添加自定义数据：

```html
<div class="user" data-user-id="123" data-username="john" data-email="john@example.com">
    John Doe
</div>
```

在 JavaScript 中可以通过 `dataset` 属性访问这些自定义数据：

```javascript
const userDiv = document.querySelector('.user');
console.log(userDiv.dataset.userId);      // 输出: 123
console.log(userDiv.dataset.username);  // 输出: john
console.log(userDiv.dataset.email);     // 输出: john@example.com
```

## 5. 标签最佳实践

### 5.1 语义化优先

- 优先使用语义化标签，如 `<header>` 而非 `<div class="header">`
- 为每个页面使用唯一的 `<h1>` 标签
- 使用合适的标题层级，不要跳过级别
- 使用 `<main>` 标签标记页面的主要内容

### 5.2 代码简洁

- 不要使用不推荐的标签和属性
- 避免不必要的嵌套
- 关闭所有标签（包括自闭合标签）
- 使用小写标签名（虽然 HTML 不区分大小写，但小写是最佳实践）

### 5.3 可访问性

- 为所有图像添加 `alt` 属性
- 为表单控件添加 `<label>` 标签
- 使用 `aria-*` 属性增强可访问性
- 确保表单控件有适当的 `name` 和 `id`
- 为链接添加有意义的文本

### 5.4 性能优化

- 减少 DOM 元素数量
- 优化图像大小和格式
- 避免使用内联样式
- 延迟加载非关键资源

### 5.5 兼容性

- 避免使用实验性标签和属性
- 为不支持的浏览器提供降级方案
- 使用 HTML5 Shiv 或 Modernizr 处理旧浏览器兼容问题

## 6. 总结

HTML 标签是构建网页的基础，了解不同标签的用途、属性和最佳实践对于创建结构良好、语义化、可访问的网页至关重要。

本文介绍了常用的 HTML 标签，包括文档结构标签、文本内容标签、链接标签、图像标签、列表标签、表格标签、媒体标签、表单标签、语义化标签等。同时，还介绍了标签的分类、全局属性和最佳实践。

通过合理使用 HTML 标签，可以创建结构清晰、语义化、可访问、高性能的网页，为后续的 CSS 样式设计和 JavaScript 交互开发打下良好的基础。
