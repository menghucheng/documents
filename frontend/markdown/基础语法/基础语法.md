# Markdown 基础语法

## 1. 概述

Markdown 是一种轻量级标记语言，由 John Gruber 于 2004 年创建，旨在让人们能够使用易读易写的纯文本格式编写文档，然后转换成有效的 XHTML（或 HTML）文档。

Markdown 的主要特点：
- 简单易用，学习成本低
- 纯文本编写，兼容性强
- 可以转换为 HTML、PDF、Word 等多种格式
- 广泛应用于文档编写、博客写作、README 文件等场景

## 2. 标题

使用 `#` 符号表示标题，`#` 的数量表示标题级别，最多支持 6 级标题。

**语法示例**：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**显示效果**：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

## 3. 段落和换行

### 3.1 段落

Markdown 中的段落是由一个或多个连续的文本行组成，段落之间用一个或多个空行分隔。

**语法示例**：

```markdown
这是第一个段落。这是第一个段落的延续，
中间的换行不会产生新的段落，只会在同一行显示。

这是第二个段落，与第一个段落之间有一个空行。
```

### 3.2 换行

如果需要在段落内换行，可以在该行末尾添加两个或多个空格，然后按回车键。

**语法示例**：

```markdown
这是第一行，末尾添加两个空格  
这是第二行，与第一行在同一段落内。
```

**显示效果**：

这是第一行，末尾添加两个空格  
这是第二行，与第一行在同一段落内。

## 4. 强调

### 4.1 斜体

使用 `*` 或 `_` 包裹文本表示斜体。

**语法示例**：

```markdown
这是 *斜体文本*，这也是 _斜体文本_。
```

**显示效果**：

这是 *斜体文本*，这也是 _斜体文本_。

### 4.2 粗体

使用 `**` 或 `__` 包裹文本表示粗体。

**语法示例**：

```markdown
这是 **粗体文本**，这也是 __粗体文本__。
```

**显示效果**：

这是 **粗体文本**，这也是 __粗体文本__。

### 4.3 粗斜体

使用 `***` 或 `___` 包裹文本表示粗斜体。

**语法示例**：
```markdown
这是 ***粗斜体文本***，这也是 ___粗斜体文本___。
```

**显示效果**：

这是 ***粗斜体文本***，这也是 ___粗斜体文本___。

## 5. 列表

### 5.1 无序列表

使用 `-`、`*` 或 `+` 作为列表标记，后跟一个空格，然后是列表项内容。

**语法示例**：

```markdown
- 无序列表项 1
- 无序列表项 2
  - 嵌套无序列表项 2.1
  - 嵌套无序列表项 2.2
* 无序列表
